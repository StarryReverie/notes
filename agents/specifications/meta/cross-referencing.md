# 跨文件引用规范

本文件定义 agents 文档中跨文件引用的维护规则。

## 引用格式

- 使用 Markdown 链接语法引用其他文件
- 引用使用相对路径

同目录内引用直接使用文件名：

```markdown
具体字段定义见 `frontmatter.md`。
```

跨目录引用使用完整相对路径：

```markdown
- `specifications/notes/frontmatter.md`
```

## 维护规则

文件移动或重命名时，必须同步更新所有引用该文件的位置。

操作步骤：

1. 确定被移动/重命名的文件的所有引用位置
2. 逐一更新每个引用为新路径
3. 验证无遗漏的旧引用

## 合规判定

以下情况判定为不合规：

- 存在指向已移动或已重命名文件的旧路径引用
- 引用路径指向不存在的文件
