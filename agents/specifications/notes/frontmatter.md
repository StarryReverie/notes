# Frontmatter 规范

笔记的 frontmatter 使用 YAML 格式，位于文件开头的 `---` 定界符之间。

## 必选字段

### `title`

- 类型：string
- 内容：中文标题
- 普通笔记：中文概念名（如 `极限`、`进程`、`静电场`）
- `_index.md` 文件：中文分类名（如 `数学`、`微积分`）

### `weight`

- 类型：integer
- 用途：控制排序顺序
- 值为 100 的倍数（如 100, 200, 300），便于在已有条目之间插入新条目
- `_index.md` 中可选，普通笔记中必选

### `math`

- 类型：boolean
- 值：`true` 或 `false`
- 当笔记包含任何 LaTeX 公式时设为 `true`
- 不包含任何数学内容时设为 `false`

## 可选字段

无。当前不使用其他字段。如需扩展，必须先通过 `add-specification` 工作流更新本文件。

## 格式要求

```yaml
---
title: 中文标题
weight: 100
math: true
---
```

- `---` 定界符各占一行
- 字段按 `title` → `weight` → `math` 顺序排列
- 不允许出现未在本文件中定义的字段

## 合规判定

以下情况判定为不合规：

- 缺少必选字段
- 字段类型错误（如 `weight` 不是整数、`math` 不是 boolean）
- 出现未定义的字段
- 字段顺序不符合规定
- YAML 格式错误（无法解析）
